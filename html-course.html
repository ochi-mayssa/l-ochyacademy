Introduction à HTML

HTML (Hypertext Markup Language) est le langage standard pour créer des pages web. Il permet de structurer le contenu d'une page web en utilisant des balises spécifiques qui définissent le texte, les images, les liens, et d'autres éléments.

Un document HTML est constitué de balises, chaque balise étant généralement composée de deux parties : une balise d'ouverture et une balise de fermeture.

1. Structure de Base d'un Document HTML

La structure de base d'un document HTML comprend des balises essentielles telles que <html>, <head>, et <body>.

Voici un exemple de structure de base d'une page HTML :

<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Titre de la page</title>
  </head>
  <body>
    <h1>Bienvenue sur ma page web!</h1>
  </body>
</html>

Cette structure contient la déclaration DOCTYPE, qui spécifie que le document est un fichier HTML5, et la balise <html>, qui englobe tout le contenu HTML.

2. Balises HTML de Base

HTML utilise des balises pour structurer et afficher le contenu. Voici quelques balises fondamentales :

  • <h1>, <h2>, <h3>, ... <h6> : Balises de titre, utilisées pour structurer les sections de la page.
  • <p> : Balise de paragraphe, utilisée pour afficher du texte.
  • <a> : Balise de lien, utilisée pour créer des liens vers d'autres pages.
  • <img> : Balise d'image, utilisée pour intégrer des images.

Voici un exemple de code HTML avec des balises de titre, paragraphe, et lien :

<h1>Mon Premier Titre</h1>
<p>Voici un paragraphe avec un lien vers un autre site.</p>

Corrections des Exercices

Correction Exercice 1

Voici la solution pour créer une page HTML avec un titre, un paragraphe et un lien :

<html>
  <head>
    <title>Ma Première Page</title>
  </head>
  <body>
    <h1>Bienvenue sur ma page</h1>
    <p>Cliquez sur ce lien pour en savoir plus.</p>
  </body>
</html>

Correction Exercice 2

Voici la solution pour créer une liste à puces :

<ul>
  <li>Premier élément</li>
  <li>Deuxième élément</li>
  <li>Troisième élément</li>
</ul>

Correction Exercice 3

Voici la solution pour créer un formulaire simple :

<form action="/submit" method="post">
  <label for="name">Nom</label>
  <input type="text" id="name" name="name">

  <label for="email">Email</label>
  <input type="email" id="email" name="email">

  <button type="submit">Envoyer</button>
</form>
