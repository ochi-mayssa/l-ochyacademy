Introduction à PHP

PHP est un langage de script côté serveur largement utilisé pour la création de sites Web dynamiques. Il peut être intégré dans des pages HTML et interagir avec des bases de données.

Dans ce cours, nous allons aborder les bases de PHP, la syntaxe, les variables, les opérateurs, les boucles, les fonctions, et bien plus encore.

1. Syntaxe de base

PHP utilise une syntaxe très simple. Les balises PHP sont définies par <?php ... ?>. Voici un exemple basique :

<?php
echo "Bonjour, le monde!";
?>

Dans cet exemple, nous utilisons la fonction echo pour afficher du texte à l'écran.

2. Variables et Types de données

En PHP, une variable commence par le signe dollar $, suivi du nom de la variable. Les types de données les plus courants sont :

  • string - chaîne de caractères
  • int - entier
  • float - nombre à virgule flottante
  • boolean - valeur vrai ou faux

Exemple :

<?php
$nom = "John";
$age = 25;
$prix = 19.99;
$estMajeur = true;
echo "Nom: $nom, Age: $age, Prix: $prix, Majeur: $estMajeur";
?>

Exercices

Exercice 1 - Formulaire et affichage

Énoncé : Créez un formulaire avec deux champs : prénom et âge. Lorsque l'utilisateur soumet le formulaire, affichez un message personnalisé avec son prénom et son âge.

Exercice 2 - Calcul de l'âge

Énoncé : Créez un formulaire permettant à l'utilisateur d'entrer sa date de naissance. Calculez et affichez son âge en années.

Exercice 3 - Vérification de la parité

Énoncé : Créez un formulaire où l'utilisateur saisit un nombre. Le script doit vérifier si ce nombre est pair ou impair et afficher le résultat.

Corrections des Exercices

Correction Exercice 1

Le formulaire et le code PHP pour cet exercice sont les suivants :

<form method="POST" action="exercice1.php">
  Prénom: <input type="text" name="prenom"><br>
  Âge: <input type="text" name="age"><br>
  <input type="submit" value="Envoyer">
</form>
<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $prenom = $_POST["prenom"];
    $age = $_POST["age"];
    echo "Bonjour, $prenom! Vous avez $age ans.";
}
?>

Correction Exercice 2

Voici la solution pour calculer l'âge :

<form method="POST" action="exercice2.php">
  Date de naissance: <input type="date" name="dob"><br>
  <input type="submit" value="Calculer l'âge">
</form>
<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $dob = $_POST["dob"];
    $age = date_diff(date_create($dob), date_create('today'))->y;
    echo "Vous avez $age ans.";
}
?>

Correction Exercice 3

Voici la solution pour vérifier la parité :

<form method="POST" action="exercice3.php">
  Nombre: <input type="text" name="nombre"><br>
  <input type="submit" value="Vérifier">
</form>
<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $nombre = $_POST["nombre"];
    if ($nombre % 2 == 0) {
        echo "$nombre est un nombre pair.";
    } else {
        echo "$nombre est un nombre impair.";
    }
}
?>
