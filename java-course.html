Introduction à Java

Java est un langage de programmation orienté objet créé par Sun Microsystems dans les années 1990. Il est aujourd'hui l'un des langages les plus populaires pour le développement d'applications web, mobiles et de bureau.

Java fonctionne selon le principe du "write once, run anywhere", ce qui signifie que le code Java peut être exécuté sur n'importe quelle machine disposant d'une machine virtuelle Java (JVM).

1. Structure d'une application Java

Une application Java de base consiste principalement en une classe contenant une méthode principale public static void main(String[] args). Voici un exemple d'application de base :

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Bonjour, monde!");
  }
}

Ce programme affiche le texte "Bonjour, monde!" dans la console.

Correction Exercice 2

Voici le code pour calculer la somme de deux nombres :

import java.util.Scanner;

public class Somme {
  public static void main(String[] args) {
    Scanner scanner = new Scanner(System.in);
    System.out.print("Entrez le premier nombre: ");
    int num1 = scanner.nextInt();
    System.out.print("Entrez le second nombre: ");
    int num2 = scanner.nextInt();
    System.out.println("La somme est: " + (num1 + num2));
  }
}

Correction Exercice 3

Voici le code pour vérifier si un nombre est pair ou impair :

import java.util.Scanner;

public class PairImpair {
  public static void main(String[] args) {
    Scanner scanner = new Scanner(System.in);
    System.out.print("Entrez un nombre: ");
    int num = scanner.nextInt();
    
    if (num % 2 == 0) {
      System.out.println("Le nombre est pair.");
    } else {
      System.out.println("Le nombre est impair.");
    }
  }
}

Correction Exercice 4

Voici le code pour trouver le plus grand nombre :

import java.util.Scanner;

public class PlusGrand {
  public static void main(String[] args) {
    Scanner scanner = new Scanner(System.in);
    System.out.print("Entrez le premier nombre: ");
    int num1 = scanner.nextInt();
    System.out.print("Entrez le second nombre: ");
    int num2 = scanner.nextInt();
    
    if (num1 > num2) {
      System.out.println("Le plus grand nombre est: " + num1);
    } else {
      System.out.println("Le plus grand nombre est: " + num2);
    }
  }
}
